#### 2. BrowserAgent.cs —— 实现默认浏览器检测

文件路径：`Agents/BrowserAgent.cs`

当前：actionObject 为空时返回 null。
改为：actionObject 为空时打开默认浏览器。

```csharp
using System.Runtime.InteropServices;
using Microsoft.Win32;

// 在 BrowserAgent 类的最后添加一个私有方法
private static string? FindDefaultBrowser()
{
    // Try Edge first (always available on Windows)
    string[] paths =
    [
        @"C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe",
        @"C:\Program Files\Microsoft\Edge\Application\msedge.exe",
    ];

    foreach (var path in paths)
    {
        if (File.Exists(path))
            return path;
    }
    return null;
}
```

并且修改 TryMatchAsync 方法：当 actionObject 为空时，不走 `return null`，而是：
```csharp
// 替换最后的 return null:
if (!string.IsNullOrEmpty(actionObject))
{
    // ... 搜索逻辑（保持现有代码） ...
}

// 打开浏览器（无搜索词时）
var browserPath = FindDefaultBrowser();
if (browserPath != null)
{
    return Task.FromResult<AgentMatchResult?>(new AgentMatchResult
    {
        Success = true,
        AgentType = AgentType.Browser,
        ActionType = SkillActionType.run_program,
        ActionParams = new() { ["path"] = browserPath },
        Reply = "正在打开浏览器..."
    });
}

return Task.FromResult<AgentMatchResult?>(null);
```

**注意：** 添加 `using Microsoft.Win32;` 和 `using System.Runtime.InteropServices;` 在文件顶部。
